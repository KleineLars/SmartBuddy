Hoe SmartGrid te installeren

// 01

Chrome Web Store

Ga naar de officiële Web Store en zoek naar SmartGrid of klik direct op SmartGrid.

// 02

Klik op "Installeren"

Klik simpelweg op de downloadknop en sta vervolgens de vereiste machtigingen toe in je browser.

// 03

Start je Smartschool

Navigeer naar je eigen Smartschool-pagina en vernieuw deze. Gefeliciteerd, je reis is begonnen!
