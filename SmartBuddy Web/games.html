Welkom bij SmartBuddy Games!

Om toegang te krijgen tot de veilige multiplayer games en AI-chatfuncties, moet je eenmalig een Gemini API-sleutel instellen.

Hoe maak je een API sleutel? (Duurt 1 minuut)

  1. Ga naar Google AI Studio.
  2. Log in met je Google-account.
  3. Klik op de blauwe knop "Create API key".
  4. Kies "Create API key in new project".
  5. Kopieer de gegenereerde sleutel (begint meestal met AIza...).
